## API Endpoints (v1)

This document outlines the v1 API endpoints for the MiniCart application.

### Auth Endpoints

```
POST    /api/v1/auth/register
POST    /api/v1/auth/login
POST    /api/v1/auth/logout
POST    /api/v1/auth/refresh-token
POST    /api/v1/auth/forgot-password
POST    /api/v1/auth/reset-password
GET     /api/v1/auth/email/verify/{id}/{hash}
POST    /api/v1/auth/email/verification-notification
```

### User Endpoints

```
GET     /api/v1/user/profile
PUT     /api/v1/user/profile
PUT     /api/v1/user/password
GET     /api/v1/user/addresses
POST    /api/v1/user/addresses
GET     /api/v1/user/addresses/{id}
PUT     /api/v1/user/addresses/{id}
DELETE  /api/v1/user/addresses/{id}
GET     /api/v1/user/orders
GET     /api/v1/user/wishlist
POST    /api/v1/user/wishlist/{productId}
DELETE  /api/v1/user/wishlist/{productId}
```

### Product & Category Endpoints

```
# Public Product Routes
GET     /api/v1/products
GET     /api/v1/products/search
GET     /api/v1/products/{slug}

# Public Category Routes
GET     /api/v1/categories
GET     /api/v1/categories/{slug}
GET     /api/v1/categories/{slug}/products

# Admin Product Routes
POST    /api/v1/products (admin)
PUT     /api/v1/products/{id} (admin)
DELETE  /api/v1/products/{id} (admin)
POST    /api/v1/products/{id}/images (admin)
DELETE  /api/v1/products/{id}/images/{imageId} (admin)
```

### Cart Endpoints

```
GET     /api/v1/cart
POST    /api/v1/cart/items
PUT     /api/v1/cart/items/{itemId}
DELETE  /api/v1/cart/items/{itemId}
POST    /api/v1/cart/apply-coupon
DELETE  /api/v1/cart/remove-coupon
```

### Order Endpoints

```
POST    /api/v1/orders/checkout
GET     /api/v1/orders
GET     /api/v1/orders/{orderNumber}
POST    /api/v1/orders/{orderNumber}/cancel
```

### Payment Endpoints

```
POST    /api/v1/payments/process
GET     /api/v1/payments/{orderId}/status
```

### Review Endpoints

```
GET     /api/v1/products/{productId}/reviews
POST    /api/v1/products/{productId}/reviews
PUT     /api/v1/reviews/{reviewId} (owner)
DELETE  /api/v1/reviews/{reviewId} (owner or admin)
```

---

### Admin Endpoints

```
# User Management
GET     /api/v1/admin/users

POST    /api/v1/admin/users/{id}/assign-roles

# Order Management

POST    /api/v1/admin/orders/{id}/shipping

# Product Management

# Payment Management
```
